# Веб экзамен

## Клонируйте проект себе на компьютер
```sh
git clone https://github.com/MaxWarAxe/web-exam.git
```

## Для запуска проекта потребуются
 
```sh
Установить npm 
```

```sh
Установить и запустить docker 
```

### Переход в директорию

```sh
cd front
```
### Установка зависимостей

```sh
npm install
```

### Запуск

```sh
npm run dev
```
